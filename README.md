# https://seankrail.dev/

## One time setup

Install pnpm [using corepack](https://pnpm.io/installation#using-corepack):

```sh
corepack enable pnpm
```

Install dependencies

```sh
pnpm install
```

## Develop

```sh
pnpm run dev
```

## Preview production build

```sh
pnpm run build --mode development
pnpm run preview
```

## Build for production

```sh
pnpm run build
```
